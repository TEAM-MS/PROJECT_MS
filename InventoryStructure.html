Inventory System Structure

1. Core Components

InventoryManager

인벤토리 시스템의 핵심 관리자 클래스

- inventorySlots: List<InventorySlot>
- currentInventory: Inventory
+ Init(): void
+ Clear(): void
+ AddItem(itemData: ItemData): void
+ RemoveItem(itemData: ItemData): void

Inventory

인벤토리 데이터를 관리하는 클래스

- items: List<Item>
- maxSize: int
+ AddItem(item: Item): bool
+ RemoveItem(item: Item): bool
+ GetItem(index: int): Item

Item

아이템 데이터를 관리하는 클래스

- itemData: ItemData
- quantity: int
+ Use(): void
+ Split(amount: int): Item

2. UI Components

InventoryUI

인벤토리 UI를 관리하는 클래스

- inventorySlots: List<InventorySlotUI>
- selectedSlot: InventorySlotUI
+ UpdateUI(): void
+ Show(): void
+ Hide(): void

InventorySlotUI

인벤토리 슬롯 UI를 관리하는 클래스

- itemIcon: Image
- quantityText: Text
+ UpdateSlot(item: Item): void
+ ClearSlot(): void

ItemPickupUI

아이템 획득 시 표시되는 UI 클래스

- itemIcon: Image
- itemName: Text
- panel: RectTransform
+ Setup(data: ItemData): void

3. Data Components

ItemData

아이템의 기본 데이터를 정의하는 클래스

- id: int
- name: string
- icon: Sprite
- maxStack: int
- description: string

4. Relationships

InventoryManager → Inventory: 인벤토리 데이터를 관리

Inventory → Item: 아이템 목록을 보유

InventoryUI → InventorySlotUI: 슬롯 UI들을 관리

Item → ItemData: 아이템의 기본 데이터 참조

5. 주요 기능

  • 아이템 추가/제거
  • 아이템 스택 관리
  • 인벤토리 UI 업데이트
  • 아이템 획득 시 UI 표시
  • 아이템 사용
